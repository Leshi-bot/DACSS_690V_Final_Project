Where have our forests gone?

Sidebar

This project is an analysis of world forest data according to its accessibility to the populations surrounding them. The initial data set was pulled from Kaggle, created by user Takumi Watanabe (2024), and references data pulled from The World Bank (2024). Additionally, data regarding population variation was eventually pulled from the Federal Reserve Bank of St.Louis, which itself pulls information from U.S. Bureau of Labor Statistics (2024). Similarly, socioeconomic data presented by World Economic Forum user Avery Koop (2022) was also extracted. These data assisted in further analyzing its effects on deforestation and its relationships with local communities. Lastly, map data was pulled from Esri (2022) to create a global visual that meticulously compared deforestation between every country for which forest data existed.

You can find the code for this dashboard here: https://github.com/Leshi-bot/DACSS_690V_Final_Project
